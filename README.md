# STAAD-blocker-repo
Block Bentley STAAD license servers from detecting your cracked installation.

To use:

1. Run staad_hostfile.bat as administrator.
2. Open powershell as administrator.
3. Type: powershell -ExecutionPolicy Bypass -File "Filepath\STAAD_inbound.ps1"
4. Relaunch powershell as administator and type: powershell -ExecutionPolicy Bypass -File "Filepath\STAAD_outbound.ps1"
